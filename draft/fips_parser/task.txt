На базе fips_parser.md мне нужно разработать команду, которая при запуске python manage.py update_ip будет парсить и заносить в базу данных (IPObject) данные по типам РИД, переходя по ссылке из поля publication_url:

1. Изобретение (ip_type == invention):
- Реферат (из блока с id='Abs') для записи в поле abstract 
- Формула (из раздела "Формула изобретения") для записи в поле claims
- Статус (из таблицы статусов) для обновления поля actual (если строго "действует", то True, иначе - False)

2. Полезная модель (ip_type == utility-model):
- Реферат (из блока с id='Abs') для записи в поле abstract 
- Формула (из раздела "Формула изобретения") для записи в поле claims
- Статус (из таблицы статусов) для обновления поля actual (если строго "действует", то True, иначе - False)

3. Промышленный образец (ip_type == industrial-design):
- Статус (из таблицы статусов) для обновления поля actual (если строго "действует", то True, иначе - False)

4. Топологии интегральных микросхем (ip_type == integrated-circuit-topology):
- Реферат (из блока с id='Abs') для записи в поле abstract 

5. Программы для ЭВМ (ip_type == computer-program):
- Реферат (из блока с id='Abs') для записи в поле abstract 
- Язык программирования (сразу после строки "<b>Язык программирования: </b>" в кавычках) для записи в поле programming_languages

6. Базы данных (ip_type == database):
- Реферат (из блока с id='Abs') для записи в поле abstract
- Система управления базами данных (сразу после строки "<b>Вид и версия системы управления базой данных: </b> в кавычках") для записи в поле dbms
